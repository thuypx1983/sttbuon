This plugin adds social sharing icons to each image in your posts.

== Description ==

Cool Image Share is a must-use plugin for any site with images in posts. It adds social icons to each image in your posts. And it doesn't just display the icons - it does that beautifully with different CSS3 effects, which attracts the attention of your visitors and makes them want to use the icons for sharing your posts.

<h4>Features:</h4>

* <strong>Auto wrapping:</strong> There's no need to wrap the images in any tags or classes, the plugin will do it for you. 
* <strong>Position:</strong> Top Left, Top Right, Bottom Left, Bottom Right, and Center.
* <strong>Orientation:</strong> The icons panel can be displayed horizontally or vertically.
* <strong>Icons Size:</strong> You can set any preferred icon size.
* <strong>Social networks:</strong> Facebook, Twitter, Google Plus, Pinterest, Reddit, Linkedin, Tumblr, Odnoklassniki, VKontakte and LiveJournal.
* <strong>Minimum image size:</strong> You can prevent the icons panel from being displayed on small images.
* <strong>Reduce icons:</strong> The plugin can automatically reduced icon size if the panel does not fit in the image.
* <strong>Icons animation:</strong> Icons can appear and hide with 30+ different CSS3 effects (provided by Animate.css library).
* <strong>Hover effects:</strong> There are 10 different hover effects when hovering the image or the social icons.
* <strong>Order:</strong> You can change the order of social icons.
* <strong>Icons themes:</strong> The plugin contains 10 different themes for the icons.
* <strong>Cross-browsing:</strong> The plugin works in all modern browsers.

and many other features!

The plugin works correctly with any images and any align, margin and padding values.

The plugin is completely free, all the functionality is immediately available, there is no "Pro" version!

If you liked my plugin, please <strong>rate</strong> it.
 

== Installation ==

1. Upload <strong>cool-image-share</strong> folder to the <strong>/wp-content/plugins/</strong> directory.
2. Activate the plugin through the <strong>Plugins</strong> menu in WordPress.
3. That's all.

Plugin settings are available in "<strong>Settings\Cool Image Share</strong>".

== Frequently Asked Questions ==

= How do I remove the icons panel from specific images? =

Just open the image for editing and add a <strong>no-share</strong> CSS class.

= How do I specify the post title and description? =

Social networks display the title and description from meta tags, so I recommend using an SEO plugin to set them.

= Does the plugin support localization? =

Yes, please use [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/cool-image-share).
